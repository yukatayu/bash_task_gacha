# bash_task_gacha
次にやることを決めるスクリプト **（やっつけ）**

`1_study.txt` ～ `5_life.txt` に適宜したいことを置きます。  
出現率を増やしたいときは複数回書きます。

使い方

| コマンド | 説明 |
| --- | --- |
| `./doing` | 今のタスクを表示 |
| `./done`  | 今のタスクを完了 |
| `./refresh` | ガチャの中身を補充 |
| ` > tasklist.txt` | ガチャの中身をクリア (bash) |
| `./postpone` | 今のタスクを後回しにする |
| `./tasklist` | 今残っているタスク(ガチャの中身)を表示する |
| `less done.txt` | タスクを終わらせた履歴を表示する |
